Draggable.js

jQuery draggability without jQuery UI

Using Draggable.js is as easy as $(element).draggable();

To eliminate confining the draggable element to its parent, set contain: false

Draggable elements can be constrained to either the x or y axis:
Constrain to vertical: $(element).draggable({ horizontal: false });

Constrain to horizontal: $(element).draggable({ vertical: false });

Remove 'draggableness' by triggering 'dragremove': Toggle Draggable
